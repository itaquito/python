# 🔄 Ciclo while

Sirve para iterar un bloque de código indefinidamente siempre y cuando se cumpla la condición previamente establecida.

## ✏️ Ejemplos

```python title="Ejemplo de un ciclo for"
cont = 0

while cont <= 5:
	print("Mundo")
	cont += 1

# Este código imprime:
# Mundo
# Mundo
# Mundo
# Mundo
# Mundo
# Mundo
```
